Wallet Authentication API

Sign-in with wallet (SIWE) authentication. Obtain a nonce-challenge message, sign it client-side, then verify the signature to receive a JWT for subsequent requests.

BASE URL https://wallet-auth-api.onrender.com

Authentication flow

01
Request nonce
POST /auth/nonce
02
Sign message
in wallet client
03
Verify signature
POST /auth/verify
04
Use JWT token
